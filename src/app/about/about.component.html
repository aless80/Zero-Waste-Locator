Aim of this app

Aim of this app

The aim of this app is to let users search and log stores selling Zero Waste or Less Waste products. This app uses Google Maps to search for stores, and will allow you to save them together with their products so that you or other users can look them up.

Technical details

Technical details

This is a MEAN (MongoDB, Express, Angular CLI 7.0.6, Node) app using the Google Maps API to show store locations on a google map. On the back-end a server is based on Node.js/Express server. Mongoose ODM is used to connect Node.js and the MongoDB database. It provides a schema based solution to organize models and running CRUD operations. JSON web tokens are used for user authentication. The front-end was built using Angular-CLI 7 and Bootstrap. The code is in this GitHub page: gmaps-ng7

About Zero Waste

Zero Waste is a lifestyle philosophy that encourages the redesign of resource life cycles so that all products are reused. The goal is for no trash to be sent to landfills, incinerators, or the ocean. Zero Waste is for me about reducing consumption, in particular of plastic products
